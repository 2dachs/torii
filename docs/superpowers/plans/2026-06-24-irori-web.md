> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Keep `irori/` intact. Add `packages/core/` for shared cost/routing logic, `irori-web/` for Next.js UI, and Supabase migrations/functions under `irori-web/supabase/`.

**Tech Stack:** Next.js App Router, TypeScript, Tailwind CSS, Supabase Auth/Postgres/RLS/Vault/Edge Functions.
